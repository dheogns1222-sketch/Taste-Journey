# 🍽️ Taste Journey

> 커플 맛집 투어 기록 서비스

우리 둘만의 맛집 투어를 기록하고, 추억을 함께 쌓아가는 공간입니다.

## 📌 프로젝트 소개

Taste Journey는 커플이 함께 방문한 맛집을 기록하고, 사진과 평점, 메모를 남길 수 있는 웹 서비스입니다. 단순한 리뷰를 넘어, 우리만의 맛집 지도를 만들어가는 경험을 제공합니다.

## ✨ 주요 기능 (예정)

- 맛집 방문 기록 (날짜, 위치, 사진, 평점)
- 커플 공유 맛집 지도
- 방문 히스토리 타임라인
- 카테고리별 필터링 (한식, 양식, 카페 등)
- 나만의 맛집 리스트 큐레이션

## 🌿 브랜치 전략

| 브랜치 | 용도 |
|--------|------|
| `main` | 배포용 (안정 버전) |
| `develop` | 개발 통합 브랜치 |
| `feat/*` | 기능 개발 |
| `fix/*` | 버그 수정 |

## 📝 커밋 규칙

[Conventional Commits](https://www.conventionalcommits.org/) 규칙을 따릅니다.

```
feat:     새로운 기능 추가
fix:      버그 수정
docs:     문서 수정
chore:    빌드/설정 변경
style:    코드 포맷 변경
refactor: 리팩토링
test:     테스트 추가/수정
```
